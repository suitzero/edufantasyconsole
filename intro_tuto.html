Tutorial 3

do you know fibonacci number? you can program it pretty easily with a few lines of code. 

function fibonacci(n)
    if n<3 then
        return 1
    else
        return fibonacci(n-1) + fibonacci(n-2)
    end
end
print(fibonacci(10))
